
# YouTube Downloader Extension

## Installation
1. Open Chrome and go to `chrome://extensions`.
2. Enable **Developer mode** (toggle in top right).
3. Click **Load unpacked**.
4. Select the `dist` folder inside this directory (`.../playground/warped-glenn/extension/dist`).

## Usage
1. Make sure the local backend server is running (`npm run dev` in `server` folder).
2. Go to any YouTube video page.
3. Click the extension icon.
4. The video URL and Cookies will be auto-detected.
5. Select Quality and Download!
